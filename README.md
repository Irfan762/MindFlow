## ğŸ Getting Started

### Prerequisites

- Node.js (v18+)
- MongoDB (Compass or Atlas)

### 1. Clone the Repository

```bash
git clone https://github.com/Irfan762/MindFlow.git
cd MindFlow
```

### 2. Backend Setup

```bash
cd backend
npm install
# Create a .env file based on the provided environment variables
npm run dev
```

### 3. Frontend Setup

```bash
cd ../frontend
npm install
npm run dev
```

The application will be available at `http://localhost:5173`.
